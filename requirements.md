# AirDash

A browser extension that helps you pick the cheapest, convenient & reliable flight as possible.

## Features

Interface should be a simple yet elegant

- Source and destination dropdowns
- One way, round trip badges
- Start date and end date based on the selected trip badge
- Checkbox for non-stop flights
